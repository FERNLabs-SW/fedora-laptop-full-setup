# Contributing

Thank you for considering contributing to the Fedora Laptop Full Setup script!  

---

## How to Contribute

1. Fork the repository.

2. Create a new branch for your feature/fix:
```bash
git checkout -b my-feature-branch
```
3. Make your changes and commit with clear messages:
```bash
git commit -m "Add support for XYZ"
```
4. Push your branch to your fork:
```bash
git push origin my-feature-branch
```
5. Create a Pull Request on GitHub.

---

# Guidelines

- Only add scripts or tools relevant to Fedora laptops.
- Avoid including personal credentials or API keys.
- Test thoroughly before submitting changes.
